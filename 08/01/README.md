# Chapter 8

## 8-1 SSHの概要と基本設定

#### SSHとは
```
Secure SHellはユーザー認証および通信データの内容を暗号化することにより、
安全な通信を行うソフトウェア
```

#### SSHを使用した通信の手順
```
ホスト認証（接続先のホストが正しいかどうか判断）
  ↓
ユーザー認証（パスワード認証または公開鍵暗号方式）
  ↓
暗号化した通信（これ以降の通信は共通鍵暗号方式）
```

#### SSHの用途
- リモートログイン
- リモートプログラムの実行
- ファイル転送
- SSHファイルシステム
- TCP/IPポートフォワーディング

#### SSHを使用してリモートログインする
```
% ssh -l ユーザー名　ホスト名
```

#### 鍵ペアを作成する
```
% ssh-keygen -t タイプ
```
